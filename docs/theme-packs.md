# Theme Packs

Theme packs let OpenClaw present the same governed-task kernel through different narrative layers.

## Kernel rule

Theme packs are display-only.

They do not change:

- task state machine
- approval rules
- risk policy
- dispatch routing
- Feishu intake split
- gateway methods
- task schema
- audit logic

What it is good at expressing:

- approval
- hierarchy
